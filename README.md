# 🚀 Cognifyz SaaS App

A full-stack SaaS-style web application with authentication, AI chatbot, admin panel, and analytics dashboard.

## 🌐 Live Demo

* 🔗 Frontend: https://cognifyz-three.vercel.app
* 🔗 Backend API: https://cognifyz-2.onrender.com

## 🧠 Features

* 🔐 User Authentication (Login / Signup)
* 📊 Analytics Dashboard
* 💬 AI Chatbot (API integrated / mock AI support)
* 👑 Admin Panel
* 🎨 Modern SaaS UI (Sidebar + Glassmorphism)
* 📱 Fully Responsive Design

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* Axios
* Framer Motion

### Backend

* Node.js
* Express.js
* MongoDB (Atlas)
* JWT Authentication

## ⚙️ Environment Variables

### Backend (.env)
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret
OPENAI_API_KEY=your_key (optional)


### Frontend (Vercel)

VITE_API_URL=https://cognifyz-2.onrender.com

## 🚀 Run Locally

### Backend
cd backend
npm install
node server.js


### Frontend
cd frontend
npm install
npm run dev
